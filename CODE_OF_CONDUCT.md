# Code of Conduct

Thor follows a simple standard: be respectful, constructive, and safety-minded.

## Expected Behavior

- Treat others with respect.
- Discuss security tooling responsibly.
- Respect legal and ethical boundaries.
- Help newcomers learn safe practices.

## Unacceptable Behavior

- Harassment, threats, or discrimination.
- Encouraging unauthorized access or abuse.
- Sharing stolen data or secrets.
- Adding features intended for stealth, persistence, or bypassing controls.

Project maintainers may remove content or contributions that violate this code.
